## About

This repository acts as a central hub for ProjectKorra addons, gathering independently built bending moves, side plugins, and extension modules in one accessible location. Whether you operate a bending-focused Minecraft server or contribute to the ProjectKorra ecosystem, you'll find an expanding library of ready-to-deploy content that enhances core bending mechanics with new elements, sub-abilities, and gameplay features.

The collection emphasizes community-driven submissions and consistent updates. Each addon is sorted into categories for straightforward browsing, and the hosted GitHub Pages site offers a clean interface for discovery and download. Server operators can quickly pinpoint compatible addons, while developers can submit their own work to reach a broader audience.

## Getting Started

Clone the repository to access all addons locally, or browse the GitHub Pages site for a clickable interface:

```bash
git clone https://github.com/coopernoah1993/pk-script-hub-addons.git
cd PK-Addons
```

Navigate the `addons/` directory to find individual plugin JARs and configuration files. Place your chosen addons in the server's `plugins/` folder alongside ProjectKorra, then restart or reload.

## Frequently Asked Questions

**How frequently is the collection updated?**  
New addons are added as the community submits them and as existing addons receive updates. Review the repository's commit history for the latest changes.

**Can I modify an addon's configuration?**  
Absolutely. Most addons include their own config.yml file where you can adjust abilities, cooldowns, and permissions.

**Will these addons work with older ProjectKorra versions?**  
Most addons target recent ProjectKorra releases. Check each addon's documentation for specific version requirements.

**How do I submit my own addon?**  
Open a pull request with your addon files, or follow the contribution guidelines located in the `docs/` folder.

**Where are the addons stored locally?**  
All addon files reside in the `addons/` directory. The GitHub Pages site offers an alternative way to browse and download without cloning the full repository.
